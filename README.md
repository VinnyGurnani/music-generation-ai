# 🎵 AI Music Generation

## 📌 Overview

This project demonstrates how Artificial Intelligence can learn musical patterns from MIDI files and generate new music automatically.

The model is trained on MIDI music data and creates new note sequences, which are then converted into a playable MIDI file.

## 🚀 Features

* Extracts notes from MIDI files
* Preprocesses musical data
* Trains an AI model on note sequences
* Generates new musical notes
* Creates a playable MIDI output file

## 🛠️ Technologies Used

* Python
* TensorFlow
* NumPy
* Music21
* MIDI Files

## 📂 Project Structure

dataset             # MIDI dataset files
preprocess.py       # Extracts notes from MIDI files
train_model.py      # Trains the AI model
generate_music.py   # Generates new note sequences
save_music.py       # Creates MIDI output file

## ⚙️ How It Works

1. MIDI files are collected as training data.
2. Notes are extracted and preprocessed.
3. The AI model learns musical patterns from the dataset.
4. The trained model generates new note sequences.
5. Generated notes are converted into a MIDI file.

## 🎼 Output

The project generates a new MIDI file containing AI-generated music based on learned musical patterns.
